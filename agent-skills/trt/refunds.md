# TRT Refunds

## Refund Requests
- Set confidence_score <= 0.55
- Escalate to patient care team
- Use honest language: "I've passed your refund request to our patient care team"
- Do NOT claim to have processed the refund
- Do NOT ask customer to make decisions - let the team handle it

## When to Fetch Order Data
- Always fetch order history for refund requests
- Reference specific order details in internal note
- Include order number and items in escalation
