# TRT Subscriptions

## Subscription Queries
- Check subscription status from patient context
- Active subscriptions include FREE monitoring blood tests
- Provide next payment date when available

## Cancellation Requests
- Set confidence_score <= 0.55
- Escalate to patient care team
- Do NOT process cancellations directly
- Use honest language about what you're doing

## Subscription Changes
- Dose adjustments: escalate to clinical team
- Payment issues: escalate to patient care team
- Pause requests: escalate to team
