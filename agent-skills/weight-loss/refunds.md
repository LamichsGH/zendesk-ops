# Weight Loss Refunds

## Refund Requests
- Set confidence_score <= 0.55
- Escalate to patient care team
- Use honest language: "I've passed your refund request to our patient care team"
- Do NOT claim to have processed the refund
- Always fetch order history for refund context

## Replacement Kit Requests
- Acknowledge with empathy
- Explain free replacement policy from KB
- Use honest language: "I've requested a replacement - our team will arrange this"
- Set confidence_score = 0.75
