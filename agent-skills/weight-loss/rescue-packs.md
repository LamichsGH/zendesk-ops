# Weight Loss Rescue Packs

## What Are Rescue Packs?
- Starter packs containing anti-nausea and anti-diarrhoea medication
- Include Cyclizine (anti-nausea) and Loperamide (anti-diarrhoea)
- Designed to help manage common GLP-1 side effects
